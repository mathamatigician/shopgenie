🚀 **Reimagining E-Commerce: Introducing ShopGenie 🧞‍♂️ — An AI-First Conversational Shopping Experience!**

Traditional e-commerce platforms like Amazon and Flipkart require users to navigate across 5-10 different screens—searching categories, dealing with multi-step checkout forms, entering payment details, and filling out star-rating surveys. 

This process creates immense friction, especially for **elderly customers, mobile users, and visually impaired shoppers**. 

What if shopping was as simple as having a conversation? 💬

Introducing **ShopGenie** — an AI-first conversational e-commerce web platform built during the **@Amadeus** Hackathon! 🏆

### 💡 **What Makes ShopGenie Unique?**

1. **Zero-Friction Conversational Shopping**: Search products, place orders, check out, and pay entirely through natural speech or text!
2. **Automated AI Tool Calling**: 
   - `create_order`: Instant order creation & stock management.
   - `pay_order`: Automated payment processing via saved card/UPI.
   - `submit_feedback`: Real-time NLP sentiment, category & urgency analysis.
3. **Multi-Item Inventory & Alternate Recommender**: Requests items, validates stock per item, and suggests in-store alternatives for out-of-stock items item-by-item!
4. **Natural Speech & Accessibility**: Built-in Web Speech API integration (Voice Input 🎙️ + Speech Synthesis 🔊) designed for accessibility.
5. **Dynamic Item & Cart Management**: Simple natural commands to list items sorted by price (`"List Amul mango ice-creams sorted by price"`), view cart (`"show cart option"`), and update quantities (`"Update quantity to 5"`).

---

### 💻 **Tech Stack:**
- **Backend**: FastAPI, Python 3.12, Uvicorn, PyJWT
- **Database**: SQLite, SQLAlchemy ORM (Dynamic CSV Seeding Pipeline)
- **Frontend**: Vue 3, Vite, Pinia/Reactive Auth Store, Glassmorphic CSS System
- **Accessibility**: Web Speech API (SpeechRecognition & SpeechSynthesis)
